# prompt-master
一键搜索、管理你的 AI 提示词，支持全键盘高效操作，对鼠标也友好，支持一键点击复制。
<img width="1788" height="1202" alt="扩展页面1" src="https://github.com/user-attachments/assets/3e6b6338-1359-4df2-9544-5c8945da1e86" />

本插件是由于市面上的很多插件都普遍只支持键盘操作，不太利于一只手用鼠标的情况，所以在AI的帮助下开发了这款对键盘和鼠标都相对友好的插件。  
## 插件功能
1. 点击扩展图标后，会弹出提示词的管理插口，为卡片式布局。  
2. 支持提示词的导入导出，格式为json。
3. 支持记忆窗口最后停留的位置。  

## 使用方法
1. 可直接点击卡片复制提示词。也支持提示词置顶。
2. 键盘方面：点击插件窗口右上角的设置按钮，可直接跳转到快捷键设置页面，然后对「**快速打开提示词管理器**」配置即可，推荐使用Alt+E。  
3. 启动插件后，**默认激活搜索**功能，此时搜索后，按下**Tab/Enter**，即可跳转到卡片区域，然后按**tab/方向键**切换卡片。  
4. 在卡片区域时，可以**按下Enter，快速复制卡片**内容。

### 复制行为说明（需求总结）
- 点击卡片右上角 `📋` 复制按钮：复制成功后自动关闭扩展窗口。
- 点击卡片内容区域：复制成功后同样自动关闭扩展窗口。
- 关闭动作为“静默关闭”策略：尽量不主动唤醒浏览器主窗口（尤其是全局快捷键场景）。
- 复制失败时不会关闭窗口，并在控制台输出错误日志。

## 本地打包（生成 zip）
在项目根目录执行：

```bash
zip -r prompt-master.zip . -x '*.git*' -x '.idea/*' -x 'prompt-master.zip'
```

- 生成文件：`prompt-master.zip`
- 已排除：`.git`、`.idea`、旧 `prompt-master.zip`

## 注意事项：
   本插件没有联网功能，所有数据均在本地。，所以建议定时导出备份文件到本地。  

### 自动备份
- 触发时机：每次启动浏览器时自动备份当前提示词；数据未变化则跳过，减少下载提示。
- 备份位置：浏览器下载目录下 `prompt-master/backups/`（文件名如 `prompts_backup_YYYYMMDD_HHMMSS.json`）。
- 保留数量：自动保留最近 5 个备份，超出会删除最旧的文件。
- 手动导入/导出仍然可用，建议在大规模编辑前先手动导出。  

### 数据存储与找回
- 存储位置：提示词保存在浏览器的 `chrome.storage.local`（本地用户配置目录下，由浏览器管理），不会自动同步到云端。
- 丢失风险：清空浏览器数据、卸载扩展或更换设备时，数据会被移除。
- 备份方法：在插件界面右上角点击「导出」，会生成 `prompts_backup.json`；建议定期备份到安全位置或云盘。
- 恢复方法：点击「导入」，选择之前导出的 `prompts_backup.json` 即可恢复。
- 升级/重新加载扩展：只要不清空浏览器数据，提示词会自动保留；若担心丢失，先导出备份再操作。

## Electron 桌面版（推荐）

仓库新增了 `electron-app/`，用于桌面端实现。

运行方式（Bun）：

```bash
cd electron-app
bun install
bun run dev
```

> 注意：首次运行可能需要安装 Xcode Command Line Tools。

打包：

```bash
cd electron-app
bun run build
```

输出目录：`electron-app/dist`。
